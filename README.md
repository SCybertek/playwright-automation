# Playwright Automation

Repository containing Playwright tests and reports for practice automation exercises.

## Prerequisites

- Node.js (>=16)
- npm

## Install

```bash
npm install
```

## Run tests

Run all tests:

```bash
npx playwright test
```

Run a specific test file:

```bash
npx playwright test tests/day02/locator-object-method.spec.js
```

Run tests with npm script (if configured):

```bash
npm run test
```

## Reports

- Playwright HTML report: `playwright-report/`
- Test results folder: `test-results/`
